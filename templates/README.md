# Templates

Templates are starter artifacts that make adoption easier without forcing one exact workflow onto every project.

## Template Levels

### Repository templates

- root README
- docs index

### Rule templates

- decision record
- problem record
- dependency inventory
- verification checklist
- exception record

### Composition templates

- core-only starter
- core + profile starter
- core + profile + adapter starter
