# Task Log

Phase/Task 완료 시마다 이 디렉터리에 5-section 형식으로 기록합니다.
상세: dev-standards R2 (Task Logging).

| — | — | — |

## Format

각 task log 파일은 [`TEMPLATE.md`](TEMPLATE.md) 를 복사하여 시작합니다.

### 5 필수 섹션

1. **언어 / 내용** — 무엇을 어떤 언어로 작성했나
2. **문제** — 발생한 이슈 (없으면 "없음" 명시)
3. **분석** — 근본 원인
4. **해결방안** — 제안/적용한 방법
5. **결과** — 검증 결과 + 커밋 해시
