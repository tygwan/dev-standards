# P4 — Public Writing Profile

## Use when

Activate this profile when the project needs structured outward-facing communication such as public case studies, portfolio pages, or technical essays.

## Purpose

This profile keeps external communication guidance separate from the core engineering standard.

## Adds on top of core

- public artifact structure
- audience-aware explanation standards
- presentation guidance for outcomes and decisions

## Why this is not core

Public writing is valuable, but it is not a universal requirement of every engineering project.

## Typical activation signals

- recruiting or portfolio use
- public technical communication
- external project storytelling

## Typical non-goals

- requiring every project to produce public narratives
- mixing communication craft rules into engineering baseline rules
