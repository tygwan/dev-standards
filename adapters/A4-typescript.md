# A4 — TypeScript Adapter

## Use when

Activate when TypeScript is a primary language in the project.

## Operationalizes

- configuration and environment
- verification
- interfaces and contracts

## Adds

- TypeScript ecosystem guidance for build, typecheck, and package surfaces
- how type contracts and runtime contracts relate
- how generated outputs and build artifacts are typically separated

## Does not define

- one framework
- one bundler
- one monorepo tool
