# A2 — Claude Code Adapter

## Use when

Activate when Claude Code is an active engineering tool in the project workflow.

## Operationalizes

- AI-assisted development profile
- documentation and discoverability
- change management

## Adds

- how project context is surfaced to Claude Code
- how long-lived behavioral guidance is stored if the team wants that pattern
- how Claude-specific workflows relate to project documentation

## Does not define

- a universal AI policy
- a requirement that all projects use persistent memory files
- a requirement that all teams use Claude Code
