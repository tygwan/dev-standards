# A3 — Python Adapter

## Use when

Activate when Python is a primary language in the project.

## Operationalizes

- configuration and environment
- verification
- dependency management

## Adds

- Python ecosystem packaging and environment guidance
- how tests, scripts, and runtime entrypoints are typically exposed
- how generated artifacts and virtual environment state are usually handled

## Does not define

- one package manager
- one formatter or linter
- one project layout for every Python repository
