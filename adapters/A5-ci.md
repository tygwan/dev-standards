# A5 — CI Adapter

## Use when

Activate when the project uses automated continuous integration checks.

## Operationalizes

- verification
- change management
- observability

## Adds

- how verification gates are enforced automatically
- how build or test failures become visible
- how artifacts or reports from automated checks are surfaced

## Does not define

- one CI vendor
- one pipeline layout
- one required check suite for every project
